# Corporate Bloat Index

Corporate Bloat Index is a Next.js 14 web app that visualizes public company headcount against revenue efficiency using D3.js. It includes bubble, scatter, and ranking views with filtering, sorting, and KPI summaries.

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

- One-click: import the repository into Vercel and deploy with default settings.
- CLI:

```bash
vercel deploy
```

## Data note

All company values are hardcoded static 2024 figures based on public filings and estimates.
